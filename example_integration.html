This page demonstrates how to integrate the chatbot into any website

ðŸ“‹ Integration Steps:

  1. Include the chatbot CSS file in your <head>
  2. Include the chatbot JavaScript file before closing </body>
  3. Initialize the chatbot with your API URL
  4. That's it! The chatbot will appear automatically

Code Example:

<!-- In <head> -->
<link rel="stylesheet" href="http://your-server:5000/static/chatbot.css">

<!-- Before closing </body> -->
<script src="http://your-server:5000/static/chatbot.js"></script>
<script>
    const chatbot = new ChatbotWidget({
        apiUrl: 'http://your-server:5000/api'
    });
</script>

Features:

  • Fully responsive chatbot widget
  • Auto-detects API URL from current page
  • Real-time autocomplete suggestions
  • Context-aware question suggestions
  • File upload and processing
  • Settings management
  • Download answers as files
  • Fullscreen mode
  • Minimized state with custom message

Update the API URL in the script below to match your server:

const chatbot = new ChatbotWidget({
    apiUrl: 'http://YOUR_SERVER_IP:5000/api'
});

API Endpoints Available:

  • GET /api/health - Health check
  • GET /api/greet - Get greeting
  • POST /api/query - Send query
  • POST /api/autocomplete - Get autocomplete
  • GET /api/faqs - Get FAQs
  • POST /api/upload - Upload file
  • POST /api/files/list - List files
  • POST /api/settings - Get/Update settings
