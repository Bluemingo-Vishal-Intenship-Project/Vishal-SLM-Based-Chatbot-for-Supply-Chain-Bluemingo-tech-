Embeddable AI-powered chatbot for your website

Features

  • Modern, responsive UI design
  • Auto-complete with smart suggestions
  • Tab key to autocomplete questions
  • Greeting message on load
  • FAQ suggestions after responses
  • Download answers functionality
  • Settings for path configuration
  • Works on any webpage via API
  • Fully customizable

Quick Start

Add this single line to your HTML to embed the chatbot:

<script src="https://your-domain.com/static/chatbot-embed.js" data-api-url="https://your-api-domain.com/api"></script>

Configuration Options

You can customize the chatbot with these data attributes:

<script src="chatbot-embed.js" data-api-url="https://api.example.com/api" data-download-path="C:/Users/Downloads" data-files-folder-path="C:/Users/Documents"></script>

Manual Initialization

For more control, you can initialize manually:

<script src="static/chatbot.js"></script> <link rel="stylesheet" href="static/chatbot.css"> <script> const chatbot = new ChatbotWidget({ apiUrl: 'https://api.example.com/api', downloadPath: 'C:/Users/Downloads', filesFolderPath: 'C:/Users/Documents' }); </script>

API Endpoints

The chatbot uses these API endpoints:

  • GET /api/greet - Get greeting message
  • POST /api/query - Send query and get response
  • POST /api/autocomplete - Get autocomplete suggestions
  • GET /api/faqs - Get all FAQs
  • POST /api/download - Download answer as file
  • GET/POST /api/settings - Get/update settings
Note: Make sure your API server is running and CORS is enabled for your domain.
